# GradeCalculator-MiniProgram

A WeChat Mini Program that helps students calculate course grades and GPA.

## Introduction

GradeCalculator is a WeChat Mini Program designed to help students calculate weighted course grades and GPA efficiently.

It provides a simple interface for managing course scores and evaluating academic performance.

## Features

- Calculate weighted course grades
- Calculate GPA
- Simple and clean user interface
- Lightweight and easy to use

## Tech Stack

- WeChat Mini Program
- JavaScript
- WXML
- WXSS
- JSON
